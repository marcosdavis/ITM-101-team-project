# Problem or issue to solve
-            Subscription manager?
-            Password manager

# Description
We decided as a group on a password manager that would provide more functionality than any other on the market at the time. Moving forward with this idea will provide value to a large number of people. This is going to eliminate the pain points of forgetting your password, not changing your password frequently enough, and not having a place to view or manage your passwords in a comfortable and usable environment. To make money with this idea, we will charge a monthly fee to use it.

# Features
-            Can see all your passwords in one place
-            Lists the different websites in widgets/lists
-            Linking it to multiple websites instead of just one url
-            Delete feature
-            Edit button to edit the password or url
-            Quick copy button
-            Separated by accounts
-            See how many times you reuse a password
-            Parent mode so you can see their passwords
-            Reminder to change your password
-            Flashcard password quiz
-            Password generator as a string
